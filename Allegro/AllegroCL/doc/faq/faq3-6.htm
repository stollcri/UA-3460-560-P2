Q 3.6-1) Can I use CL-HTTP with Allegro CL?

A: 3.6-1) Allegro CL 8.0 does work with CL-HTTP but code merges have not (at the time of writing) been posted in the standard locations. Therefore, getting the correct code is complicated. If you wish to use CL-HTTP with Allegro CL 8.0, please send email to support@franz.com for futher information.

CL-HTTP is the Common Lisp Hypermedia Server developed by John C. Mallery (jcma@ai.mit.edu) of the M.I.T. Artificial Intelligence Laboratory.
