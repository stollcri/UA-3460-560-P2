*jni-library*

This variable must be set to an appropriate value if the native mode of Jlinker is used. It may be a relative or an absolute pathname string. A relative pathname is conbined with the value of *jlinker-java-home*. An absolute path is used verbatim.

The path must point to the Java VM shared library.

  • On Windows, the value is typically "jre/bin/client/jvm.dll".
  • On some Linux x86 systems, we have used the value "jre/lib/i386/client/libjvm.so".
  • On some SunOS sparc systems, we have used the value "jre/lib/sparc/client/libjvm.so".
  • On some MacOSX systems, we have used "/System/Library/Frameworks/JavaVM.framework/JavaVM".

See jlinker.htm for general information on the jLinker facility.
