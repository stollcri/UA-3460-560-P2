*cl-default-special-bindings*

This variable holds an alist associating special variable symbols with forms to evaluate for binding values. When a process first starts execution its initial-bindings slot is examined for bindings to establish for the process. *cl-default-special-bindings* contains a reasonable default list which establishes a default environment with standard Common Lisp values. The contents of this list may be examined, but if you want to modify it for a particular call to make-process, it is strongly suggested you either modify a copy made with copy-alist, or just cons replacement entries onto the front of the list.

Samples of the contents of *cl-default-special-bindings* are shown in the table below. For all common-lisp package variables the initial value will be the same as the standard Common Lisp default.
