*trace-print-circle*

*print-circle* is bound to the value of this variable when trace output is being printed.

The value of this variable may be, in addition to the values usually allowed for a print control variable, the symbol :follow. That value means use the current value of cl:*print-circle*. See Allegro CL print variables can follow the CL print variable value in implementation.htm.

See also debugging.htm for general information on tracing in Allegro CL.
