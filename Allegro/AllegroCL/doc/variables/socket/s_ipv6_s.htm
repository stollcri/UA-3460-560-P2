*ipv6*

IPv6 support was added to Allegro CL 8.0.beta with an update. Get the update with sys:update-allegro. When the update is loaded, the feature :ipv6 is added to the *features* list.

This variable controls the default behavior of lookup-hostname and make-socket.

The valid values are:

  • nil: Use IPv4 addressing only (unless an IPv6 address is specified for local-host or remote-host in a call to make-socket).
  • :ipv6: Use IPv6 addressing only.
  • t: Prefer IPv6 over IPv4 for hostname lookups. Create sockets using IPv6.

See socket.htm for general information on sockets in Allegro CL.
