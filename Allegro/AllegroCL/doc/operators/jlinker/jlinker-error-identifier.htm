jlinker-error-identifier

Arguments: jlinker-error

The function was added to the jLinker facility in jlinker version 3.2.4 released as patch jlinker.001 around Dec. 6, 2002. The currently running version is returned by the function jlinker-version, with a call like (javatools.jlinker:jlinker-version). Get the patch for the new version with sys:update-allegro.

This generic function is the accessor of the dentifier slot of jlinker-error. The identifier slot holds a keyword or a string describing the error.

See jlinker.htm for more information on the jLinker facility.
