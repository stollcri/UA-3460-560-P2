sha1-file

Arguments: filespec

Using sha1-init, sha1-update, and sha1-final, compute the MD5 checksum on the contents of the file given by filespec. Returns the value of sha1-final on the implicitly created context.

See MD5, SHA1, HMAC, and RC4 support in miscellaneous.htm.
