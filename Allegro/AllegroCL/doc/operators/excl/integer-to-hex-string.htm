integer-to-hex-string

Arguments: &body integer

Converts integer to a hexadecimal string. This function is designed to be very efficient.

See hex-string-to-integer, which is the inverse of this function, and MD5, SHA1, HMAC, and RC4 support in miscellaneous.htm. See also usb8-array-to-hex-string and hex-string-to-integer.
