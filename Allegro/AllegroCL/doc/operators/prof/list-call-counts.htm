list-call-counts

Arguments: &key current-profile

Returns an alist of all functions that have been counted along with their call counts, sorted in descending order of counts. profile defaults to the current profile. If supplied, its value must be a profile saved by save-current-profile.

See runtime-analyzer.htm for general information on the runtime analyzer.
