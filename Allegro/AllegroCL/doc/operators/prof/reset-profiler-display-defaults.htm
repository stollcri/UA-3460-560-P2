reset-profiler-display-defaults

Arguments: nil

This function resets, to their initial values, the values of the following runtime analyzer display control variables:

  • *hidden-packages*
  • *hidden-functions*
  • *hidden-space-functions*
  • *significance-threshold*
  • *fractional-significance-threshold*

See runtime-analyzer.htm for general information on the runtime analyzer.
