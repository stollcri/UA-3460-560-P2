mozilla-library-path

Arguments: ide-configuration

Applies to GTK only.

This IDE configuration option will remember the current value of excl:*mozilla-library-path* in the IDE's options file.

If you have located the Mozilla GTK control that's needed for the html-widget to work on GTK and set the value of excl:*mozilla-library-path* accordingly in the IDE, then the existence of this configuration option should cause the IDE to remember the proper path in future IDE sessions.
