cg-emacs-comtab

Arguments:

Returns a comtab (command table) that contains a subset of the IDE editor keystrokes when ide:editor-mode is :emacs. This comtab could be given to a text-edit-pane in a CG application to provide the IDE editor's general text-editing keystrokes that are not specific to the IDE.

See cg-host-comtab for a table of all of the keystrokes that are provided by this comtab as well as that one.
