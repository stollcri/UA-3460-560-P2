column-width

Arguments: item-list

Returns the width of each column in pixels on single-item-list and multi-item-list controls. The value nil is arbitrarily interpreted as a width of 150 pixels. There is no maximum width.

column-width is a property of the multi-item-list and single-item-list classes.

GTK Note

This property is ignored on GTK, where the native item-list widget does not support multiple columns.
