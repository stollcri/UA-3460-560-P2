text-edit-comtab

Arguments: system

Returns the text-edit comtab of system. The current system is the value of *system*. This comtab is the parent command table of the pre-supplied editor comtabs. text-edit-comtab is a property of the system class.

See also set-event-function, which has examples of comtabs.
