positionp

Arguments: other

Returns t if the argument is a position object and returns nil if it is not. See make-position.

Warning about positions created by the with-positions and with-positions-and-boxes

positionp should not be called on objects created by the macros with-positions and with-positions-and-boxes. The return value of those functions is undefined for objects created by the macros because type information for such objects is sacrificed to achieve the non-consing efficiency of stack allocation.
