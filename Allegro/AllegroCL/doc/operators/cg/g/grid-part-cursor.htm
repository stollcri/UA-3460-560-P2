grid-part-cursor

Arguments: grid-widget column-section column-section-border-p column column-num column-border-p x row-section row-section-border-p row row-num row-border-p y

Returns the mouse cursor to use when the mouse is over some part of a grid. (It returns a cursor handle as returned from cg::open-cursor, rather than a cursor as returned from cg::make-cursor, to prevent flashing.)
