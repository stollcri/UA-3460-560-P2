horizontal-justifiction

Arguments: object

Returns a symbol indicating how text is aligned horizontally in the cells of a grid-column or in a static-scrolling-text widget. The value may be set at creation time by passing the horizontal-justification initarg, or any time later by calling (setf horizontal-justification). The value may be :left, :center, or :right. The default value is :left.

For grid-widget cells, the default cell-horizontal-justification method will use this grid-column property, but a custom cell-horizontal-justification method may override it.
