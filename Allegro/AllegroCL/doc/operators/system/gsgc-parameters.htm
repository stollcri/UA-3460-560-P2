gsgc-parameters

Arguments:

This function prints the names and values of all the gsgc parameters and switches. The values printed as 

(switch :xxx)

are switches and are set and read with gsgc-switch. The other values are parameters, which are set and read with gsgc-parameter.

See System parameters and switches in gc.htm for more information on garbage collection and gsgc parameters and switches.
