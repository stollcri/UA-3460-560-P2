presto-build-lib

Arguments: fname

In previous releases, this function copied all stub definitions for libfasl (Allegro Presto) loaded functions to the file specified by the argument. In 7.0, the Allegro Presto facility has been removed. This function is kept for backward compatibility only. It is a no-op (does nothing) and a warning is signaled if it is called. The file specified by the argument will not be created (but is no longer necessary anyway). See The Allegro Presto facility has been removed in loading.htm for further information.
