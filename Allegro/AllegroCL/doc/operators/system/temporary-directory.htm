temporary-directory

Arguments:

Returns the pathname of the temporary directory used by the current running Lisp for writing temporary files. This directory is chosen according to the following algorithms.

On UNIX: #p"/tmp/".

On Windows: the first of the following to be a valid directory is used:

  • the value of the environment variable TEMP
  • the value of the environment variable TMP
  • [current-drive]:\tmp
  • C:\ (the last presumably always exists)

The temporary directory used by a running Lisp cannot be changed. This function does not work with setf. It is informational only.
