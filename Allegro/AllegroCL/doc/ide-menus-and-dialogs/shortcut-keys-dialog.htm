Shortcut Keys

This dialog, which displays the currently-defined shortcut keys, is displayed by clicking Help | Shortcut Keys. Some of the shortcuts come from menus and dialogs and others come from the current comtab (which defines editor behavior). If there is a conflict, the comtab command overrides the other commands. Note the command: quick-symbol-info-on-space connected to the SPACE key. Note that (assuming this is in effect as you run Lisp) pressing SPACE enters a space in the editor to which you are typing and displays quick symbol info in the status bar in the Project Window.

When this dialog is selected, the File | Print command will print the entire list of key bindings, complete with each command's help string. The printed list will be in the order in which the dialog is currently sorted. See also the function comtab-report, which can either present this dialog or print the bindings as a plain text table to an arbitrary output stream.
