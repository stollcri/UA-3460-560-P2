Install Menu

The Install Menu contains commands that either download new patches to lisp itself, or download and install the latest version of particular add-on products. The commands on the menu are:

  • New Patches -- Displays the Download Patches dialog for downloading all or selected Allegro CL patches.
  • Latest AllegroCache -- Loads the :update module if needed, and then calls system.update:install-allegrocache to download and install the latest version of the AllegroCache object-oriented database software.
  • Latest AllegroGraph -- : Loads the :update module if needed, and then calls system.update:install-allegrograph to download and install the latest version of the AllegroGraph triple-store software.
