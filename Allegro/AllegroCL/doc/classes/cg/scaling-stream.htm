scaling-stream

A mixin allowing modification of stream-units-per-inch and stream-units-per-mm. Scaling-stream is the parent of the printer class but not of any of the built-in window classes. To make a scalable window, you must create a class with the cg:scaling-stream mixin and some window class as its superclasses.

GTK Note

The class scaling-stream and the function (setf stream-units-per-inch) have not yet been implemented on the GTK platform.
