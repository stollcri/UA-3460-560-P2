mime-part

This is the superclass of mime part objects. Its subclasses include mime-part-constructed. You do not instantiate this class directly.

The slot accessors are:

  • mime-part-type
  • mime-part-subtype
  • mime-part-parameters
  • mime-part-id
  • mime-part-description
  • mime-part-encoding
  • mime-part-headers
  • mime-part-parts
  • mime-part-boundary

See MIME support in imap.htm for more information.
