Printing ORBLink documentation

The ORBLink documentation comprises two parts:
  1. The vendor-independent standardized IDL/Lisp mapping.
  2. The ORBLink-specific user's manual.
All of this information is contained in the on-line documentation

The pdf file containing the mapping document is located at doc/mapping.pdf. If you are unable to read a .pdf file, you should do one of the following:

  1. Download and install the Acrobat Portable Document Format reader:

  2. Contact Franz Inc. for a hardcopy of the documentation.
